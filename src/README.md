# 🌍 AEMET Weather Data Collector - Cosunsumo_energetico

Colector de datos meteorológicos de la AEMET (Agencia Estatal de Meteorología de España)

## 📋 Requisitos previos

- Python 3.11 o superior
- API Key de AEMET (gratuita)

## 🚀 Ejecución en GitHub Codespaces

1. Abre el repositorio en GitHub
2. Haz clic en `Code` → `Codespaces` → `Create codespace on main`
3. Espera a que se configure el entorno (2-3 minutos)
4. En la terminal, ejecuta:
   ```bash
   python src/aemet_collector.py
